## About

Materialvect is a powerful web based Material 3 Expressive SVG editor.

## How to Use

1. Select folder to store projects
2. Choose a template or create a new vector
3. Start creating!

## Features

- Free and open-source
- Local first
- Progressive Web App
- Dynamic Material 3 Expressive token colors
- Google Drive backups
- Project Fugu APIs

## Development

### Prerequisites

- **Node.js** to run the project.

### Build and Run

1. Run `npm install` to install the project.
2. Configure your .env to have:

```
VITE_IDB_NAME=materialvect
VITE_IDB_VER=4
VITE_IS_BETA=true
VITE_GOOGLE_CLIENT_ID=[your Google client ID]
```

3. Run `npm run dev` to run the development server.

## Disclaimer

Materialvect only works on Chromium-based browsers that support Project Fugu APIs. The Web File System Access API is required for Materialvect to work.
